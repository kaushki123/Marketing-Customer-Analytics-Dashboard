# Marketing & Customer Analytics Dashboard

Interactive Marketing & Customer Analytics Dashboard built using Power BI to analyze customer behavior, spending patterns, campaign performance, customer demographics, and purchase channels.

## Project Overview

This dashboard provides insights into customer demographics, spending habits, marketing campaign effectiveness, and purchasing behavior. It helps businesses understand customer segments and improve marketing strategies.

## Key KPIs

* Total Customers
* Total Spending
* Average Income
* Average Age
* Campaign Acceptance Rate

## Dashboard Features

* Customer Distribution by Income Group
* Campaign Response Analysis
* Spending by Product Category
* Purchase Channel Analysis
* Customer Age Group Analysis
* Customer Demographics Insights

## Tools Used

* Power BI
* Power Query
* DAX
* Excel
